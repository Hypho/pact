# FDG — 功能依赖图
> 类型：可选规划附件（optional planning artifact）。
> 挂载点：开发者明确提供 3+ 个已知功能，并选择生成 FDG 时创建。
> 作用：确认功能开发顺序，避免上游未完成时下游开始开发。
> 存放路径：生成后保存至 .pact/specs/FDG.md

版本：v1.0 | 最后更新：[日期]

```
[基础层，无依赖]

[第二层]
├── [功能 C]（依赖：A）

```
